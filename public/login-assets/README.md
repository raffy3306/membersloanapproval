# Login Page Assets

Files placed in this folder are served from `/login-assets/`.

Use these filenames for the login page:

- `login-background.png`
- `login-background.jpg`
- `login-background.webp`
- `coop-logo.png`

The page checks the background filenames in that order. The optional `coop-logo.png` replaces the CSS fallback wordmark when present.
